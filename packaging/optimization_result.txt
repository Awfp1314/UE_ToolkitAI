5. **moviepy** (~15MB) - 视频处理

7. **其他未使用的库**

1. **numpy** (~15MB) - AI 助手记忆管理需要
2. **PyQt6** (~25MB) - UI 框架
3. **PIL** (~5MB) - 图像处理
4. **requests** (~1MB) - API Key 客户端
5. **httpx** (~1MB) - Ollama 客户端
6. **其他核心依赖**

## 恢复指南

如果需要恢复原始配置（包含所有 AI 库）：

```bash

copy packaging\ue_toolkit.spec.backup packaging\ue_toolkit.spec

# 2. 重新打包
python -m PyInstaller packaging\ue_toolkit.spec --clean
```
