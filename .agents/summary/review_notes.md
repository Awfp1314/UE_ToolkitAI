## 语言支持限制

### 已支持的语言

- **Python**: 完全支持，主要开发语言
- **JSON**: 完全支持，配置文件
- **YAML**: 完全支持，配置文件
- **Markdown**: 完全支持，文档

### 未支持的语言

- **C++**: 部分 UE 插件使用 C++，但不在本项目范围内
- **Blueprint**: UE 蓝图，通过 MCP 协议间接支持

### 影响

- 文档主要关注 Python 代码
- C++ 插件的文档需要单独维护
- 蓝图相关功能依赖外部插件
